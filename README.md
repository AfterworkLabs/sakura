# Sakura - Randomizer Web Client

## Installing Prerequistites
1. Node
  * Install from [nodejs.org](https://nodejs.org)

2. Homebrew
  * Install from [brew.sh](https://brew.sh/)

3. Yarn
```sh
brew install yarn
```
